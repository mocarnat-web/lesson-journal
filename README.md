# Lesson Journal — Deploy to Vercel

## Steps

### 1. GitHub
- Sign in to github.com as mocarnat-web
- Click + → New repository → name it `lesson-journal` → Private → Create
- Click "uploading an existing file"
- Unzip this folder and drag ALL files in
- Click Commit changes

### 2. Vercel
- Go to vercel.com → sign in with GitHub
- Click Add New Project → find lesson-journal → Import
- Click Environment Variables → add:
  - Name: NEXT_PUBLIC_PASSWORD
  - Value: your chosen password
- Click Deploy

Your app will be live in ~2 minutes!
